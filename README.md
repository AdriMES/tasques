# PROJECTE KANBAN (PRÀCTICA 4 DE 2nDE DAW)

El que tracta aquesta pràctica és de crear una pàgina funcional d'un tauler kanban amb l'opció d'afegir, eliminar i editar
tasques.

Poder canviar l'estat de la tasca cada pic que nosaltres volguem i ens ensenyi unes estadístiques de les tasques que hi ha.
