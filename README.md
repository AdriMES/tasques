# PROJECTE KANBAN (PRÀCTICA 4 DE 2nDE DAW)

El que tracta aquesta pràctica és de crear una pàgina funcional d'un tauler kanban amb l'opció d'afegir, eliminar i editar
tasques.

Poder canviar l'estat de la tasca cada pic que nosaltres volguem i ens ensenyi unes estadístiques de les tasques que hi ha.

# Kanban Board · DAW

## Què és
Aplicació Kanban per gestionar tasques amb estat (Per fer / En curs / Fet), prioritat i data límit.

## Guia d’ús
- Crear tasques: omplint formulari i clicar “Afegir tasca”.
- Editar tasques: clicar “Editar” a la targeta i guardar canvis.
- Canviar estat: seleccionar a la targeta.
- Filtrar i cercar: utilitzar selects i input de cerca.
- Les estadístiques s’actualitzen automàticament.

## Estructura de carpetes
- `index.html` → pàgina principal
- `css/` → fulls d’estil
- `js/` → scripts (kanban.js)
- `img/` → captures i assets
- `README.md` → documentació
- `PDF/` → procés i evidència
